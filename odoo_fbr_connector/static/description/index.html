Installation & Configuration

  1. Copy or download the odoo_fbr_connector module to your Odoo addons directory.
  2. Restart your Odoo server.
  3. Navigate to 'Apps' in Odoo, search for "FBR Connector Backend", and click 'Activate'.
  4. Configure FBR settings in Odoo (API keys, environment, etc.).
